# markup

This repository contains introductory examples and teaching material for **markup languages** we are going to use.

## Overview

- **HTML**
- **CSS**
- **TEI/XML**

## Repository structure

- [`README.md`](./README.md)  
  This overview file.

- [`HTML_intro.md`](./HTML_intro.md)  
  Introductory notes and examples for basic HTML structure and elements.

- [`CSS_intro.md`](./CSS_intro.md)  
  Introductory notes on CSS syntax and styling concepts.

- [`TEI_intro.md`](./TEI_intro.md)  
  Introduction to TEI and XML concepts for scholarly/digital editions.

- [`html_css_examples/`](./html_css_examples/)  
  Small practical examples combining HTML and CSS:
  - [`test.html`](./html_css_examples/test.html) – simple HTML example page
  - [`test_styles.css`](./html_css_examples/test_styles.css) – stylesheet used by the HTML examples
  - [`hanslick_edition_bsp.html`](./html_css_examples/hanslick_edition_bsp.html) – example edition page in HTML

- [`TEI_XML/`](./TEI_XML/)  
  XML-based example material:
  - [`tei_template.xml`](./TEI_XML/tei_template.xml) – starter template for TEI encoding
